# IkarusShell

Simple and lightweight Reverse / Bind Shell Generator written in Python.

## Features

- Reverse Shells (Bash, Python, PHP, Perl, PowerShell, Ruby...)
- Full PHP Reverse Shell (Pentest Monkey)
- Bind Shells
- Built-in Listener
- Clean Terminal Interface

## Installation

```bash
git clone https://github.com/Abdulazizalfailkawi/IkarusShell.git
cd IkarusShell
chmod +x IkarusShell.py

##Usage
./IkarusShell.py
